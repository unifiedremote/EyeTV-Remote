# EyeTV
EyeTV for Mac OS X

## Features
*  Switch channel (up, down)
*  Close Window
*  Seek fast (forward, backward)
*  Jump (backward, forward)
*  Volume (lower, raise, mute)
*  Navigate (up, down, left, right)
*  Toogle playback (play/pause)
*  Toggle fullscreen
*  Zoom
